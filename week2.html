The 3 questions to answer before any graph problem
Q1
Is the graph explicit or implicit?
Explicit: adjacency list/matrix given directly. Implicit: grid problem, word ladder, state space — you must define what a node is and what an edge is before writing any code. Most Hard graph problems are implicit. Define node and edge first.
Q2
Single source or multi-source?
Single source: start BFS/DFS from one node. Multi-source: seed ALL sources into queue simultaneously before starting BFS. Rotting oranges, walls and gates, 01 matrix — all multi-source. This distinction fixes 30% of wrong graph answers.
Q3
What do I need from the traversal?
Shortest path, unweighted → BFS
Shortest path, weighted positive → Dijkstra
Shortest path, weighted with negative → Bellman-Ford
All paths / connectivity / components → DFS
Dependency ordering → Topological sort
Dynamic connectivity → Union-Find
Multi-source BFS — the most missed insight
1
When to use it
Whenever the problem says "distance from nearest X" or "spread from multiple starting points simultaneously." Rotting oranges, walls and gates, 01 matrix, Pacific Atlantic water flow.
2
How it differs from single source
Single source: push one node, BFS expands outward. Multi-source: push ALL source nodes before starting the BFS loop. The queue starts with multiple nodes at distance 0. Everything else is identical.
// Multi-source BFS template
deque<pair<int,int>> q;
for each source (i,j):
  q.push_back({i, j}); // seed ALL sources first
  dist[i][j] = 0;

while (!q.empty()) {
  auto [r, c] = q.front(); q.pop_front();
  for each neighbor (nr, nc):
    if (dist[nr][nc] == -1) {
      dist[nr][nc] = dist[r][c] + 1;
      q.push_back({nr, nc});
    }
}
Topological sort — two variants
Kahn's BFS
When: dependency ordering, detect cycle in directed graph
Build in-degree array. Push all nodes with in-degree 0. Process queue — for each node, decrement neighbors' in-degree, push when 0. If processed count < n → cycle exists.
vector<int> indegree(n, 0);
for each edge (u,v): indegree[v]++;
queue<int> q;
for (i=0; i<n; i++) if (indegree[i]==0) q.push(i);
while (!q.empty()) {
  int u = q.front(); q.pop(); order.push_back(u);
  for (int v : adj[u])
    if (--indegree[v] == 0) q.push(v);
}
return order.size() == n; // false = cycle
DFS postorder
When: same problems, different style
DFS with visited array. Push node to result AFTER all neighbors processed. Reverse at end. Use color states (0=unvisited, 1=in-stack, 2=done) to detect cycles.
// 0=unvisited, 1=in-stack, 2=done
bool dfs(int u) {
  color[u] = 1;
  for (int v : adj[u]) {
    if (color[v] == 1) return false; // cycle
    if (color[v] == 0 && !dfs(v)) return false;
  }
  color[u] = 2; order.push_back(u);
  return true;
}
Dijkstra — when and how
When
Weighted graph, positive weights, shortest path
Swim in rising water, network delay time, path with minimum effort, cheapest flights (use Bellman-Ford if k stops constraint). Key signal: "minimum cost path" + weights given.
priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
vector<int> dist(n, INT_MAX);
dist[src] = 0; pq.push({0, src});
while (!pq.empty()) {
  auto [d, u] = pq.top(); pq.pop();
  if (d > dist[u]) continue; // stale entry
  for (auto [v, w] : adj[u])
    if (dist[u]+w < dist[v]) {
      dist[v] = dist[u]+w;
      pq.push({dist[v], v});
    }
}
Implicit graph — the hardest graph skill
Rule
Always define node and edge explicitly before coding
For Bus Routes: node = bus route (not bus stop). Edge = two routes share a stop. For Word Ladder: node = word. Edge = two words differ by one letter. For Jump Game III: node = index. Edge = index ± arr[index]. Writing this down before coding prevents 80% of wrong approaches on implicit graph problems.

2
DFS — connected components
void dfs(int u, vector<bool>& vis, vector<vector<int>>& adj) {
  vis[u] = true;
  for (int v : adj[u])
    if (!vis[v]) dfs(v, vis, adj);
}
int components = 0;
for (int i=0; i<n; i++)
  if (!vis[i]) { dfs(i, vis, adj); components++; }
3
Union-Find with path compression + rank
vector<int> parent(n), rank(n, 0);
iota(parent.begin(), parent.end(), 0);
int find(int x) {
  return parent[x]==x ? x : parent[x]=find(parent[x]);
}
bool unite(int x, int y) {
  x=find(x); y=find(y); if(x==y) return false;
  if(rank[x]<rank[y]) swap(x,y);
  parent[y]=x; if(rank[x]==rank[y]) rank[x]++;
  return true;
}

Bipartite check
vector<int> color(n, -1);
bool bfs(int src) {
  queue<int> q; q.push(src); color[src]=0;
  while(!q.empty()) {
    int u=q.front(); q.pop();
    for(int v:adj[u]) {
      if(color[v]==-1){color[v]=1-color[u];q.push(v);}
      else if(color[v]==color[u]) return false;
    }
  }
  return true;
}
