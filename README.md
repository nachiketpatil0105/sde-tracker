# 🎯 SDE Interview Tracker

A beautiful, offline-first progress tracker for cracking SDE/SWE interviews.

## 🚀 How to Host FREE (Lifetime) on GitHub Pages

1. **Create a GitHub account** at https://github.com (free)
2. **Create a new repository** → name it `sde-tracker` (or anything)
3. **Upload all files** from this zip (drag & drop into GitHub)
4. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
5. Your site is live at: `https://YOUR_USERNAME.github.io/sde-tracker`

That's it! Free forever. No credit card. No expiry.

## 📁 Files
- `index.html` — The main tracker app
- `sde_revision_sheet.json` — Your problem sheet (update anytime!)
- `README.md` — This file

## ✨ Features
- ✅ Progress saved in browser IndexedDB (survives reloads)
- 🔍 Search, filter by difficulty/importance/status
- 📊 Live stats, streaks, topic progress
- 🌙 Beautiful dark UI
- 📱 Mobile friendly
- 🔗 Direct LeetCode links

## 🔄 Updating Your Problem Sheet
Just replace `sde_revision_sheet.json` with the new version and push to GitHub.
Your progress is stored separately in the browser, so it won't be lost!
